App Classifier

A multinomial bayes classifier model trained on IOS app store app information to predict a specefic app's category given its description.

Model

Naive Bayes

The model is called so because of the 'naive' assumptions it makes. The model assumes that features, in this case, the words in an app's description, are independent to one another.

Multinomial

Multinomial flavor of the Bayes classifier extends the model to include multiple classes or categories. Given a feature set the posterior probability must be calculated for each category and the largest the one with the largest probability is picked as the prediction.

Dataset

The model is trained on dataset of over 7000 apps from Apple's mobile app store. Before the dataset was used to train the model a few alterations, like removing apps with descriptions that contained one or more language that were not English and formatting text; were needed to ensure minimal error rate on predictions.
